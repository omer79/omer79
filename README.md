- **Civil3D Assistant Automation Tool (Python + Dynamo)**  
  Automation tool that runs inside Civil3D to automatically generate multiple engineering reports (design slopes, offsets, road points, centerline coordinates, non-design slopes) and export them to a structured Excel file, reducing several hours of manual work to minutes.

- **AutoCAD Data Management and Automation Tool (Python)**  
  Tool that reads polygon geometry from AutoCAD drawings, calculates directions, lengths, angles, and area, generates Arabic technical descriptions, exports coordinates, and creates KML files for Google Earth visualization.

- **Excel Automation Tool (VBA)**  
  Excel macro system that updates multiple sheets based on a selected log entry, performs calculations, selects table data automatically, and generates multiple PDF reports with the required data.

- **File Workflow Automation System (VBA)**  
  Excel-based file management automation tool for creating folder structures, batch renaming folders and files, changing file extensions, and generating full directory structure maps.

- **Civil3D Dynamo Python Scripts (Python)**  
  Collection of Dynamo scripts for Civil3D automation including quantity takeoff, data extraction, and workflow automation for engineering drawings.
